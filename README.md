# Atlas de Miologia

PWA imersivo e responsivo para estudo de Anatomia Humana - Miologia.

## Créditos
- Profª Drª Carmem Patrícia Barbosa
- Responsável tecnológica: Profª Dra Cláudia Pinheiro

## Recursos
- conteúdo organizado por regiões
- fotos por item do roteiro
- importação de imagens
- anotações por item
- marcação de estudo
- geração de PDF via impressão do navegador
- organização de imagens para material 3D
- funcionamento offline via service worker
- instalação como PWA em Android e iPhone

## Publicação no GitHub Pages
1. Envie todo o conteúdo deste projeto para um repositório.
2. Ative o GitHub Pages na branch principal.
3. Acesse a URL publicada pelo GitHub Pages para instalar o app.

## Observação
Os nomes foram mantidos conforme o roteiro enviado, inclusive onde o documento apresenta subitens e repetições de numeração.
